Detailed Description

TaskExecutor.

This class manages execution of tasks. Each task is a callback, i.e. method of class. Task can be ordered by level of execution and can be combined in single pool via class TaskExecutor::SingleTaskPool. Severel task pools can be combined into one group pool via class TaskExecutor::GroupTaskPool.
