Overview

Why

Oak engine is a open source game engine distributed via zlib license. Most of game engines have focus on render techniques. The main target of Oak is to deliver the best toolset and I hope, Oak will be a good light weighted alternative to Unity and Unreal Engine for indie games. All system code has simple ideas behind them and focused on easy to use functionality.

Features

  • Render system that supports DX11 and OpenGL
  • Powerful system for binding properties into the editor
  • Memory manoger with ability to find memory leaks
  • Controls system based on aliases
  • Font system based on stb_truetype.h
