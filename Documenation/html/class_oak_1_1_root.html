Detailed Description

Root.

This is a main object which allows to access to every subsytem of the engine. Engine supports multi logs, i.e logs from differentent entities can be targeted to a seprated log file. Logs on Windows stored in subfolder "From dd_mm_yyyy hh_mm" in directore Logs.
